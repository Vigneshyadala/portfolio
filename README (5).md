## About

A single-page, dark-themed portfolio with an animated particle background, a typing terminal intro, and scroll-reveal sections covering my background, projects, and skill set.

## Features

- 🌌 Animated canvas particle background with ambient glow
- 💻 Typing terminal animation on load
- 🎴 Project cards pulled from real GitHub repos, each with code + live demo links
- 📱 Fully responsive, accessible (keyboard focus states, reduced-motion support)
- ⚡ Zero build step — pure HTML/CSS/JS, deployable anywhere

## Sections

| Section | What's there |
|---|---|
| Hero | Profile photo, name, role, quick intro terminal |
| About | Background, core tech stack |
| Projects | 6 projects spanning AI engineering, full-stack apps, and automation |
| Skills | Software engineering, AI engineering, full-stack & product tooling |
| Contact | Email, GitHub, LinkedIn |

## Tech Stack

`HTML5` · `CSS3` · `JavaScript` (vanilla, no frameworks) · Google Fonts (Space Grotesk, Inter, JetBrains Mono)

## Run locally

```bash
git clone https://github.com/Vigneshyadala/portfolio.git
cd portfolio
# just open index.html in your browser — no build step needed
```

## Deployment

Hosted on **Netlify** (primary) at [vigneshyadala.netlify.app](https://vigneshyadala.netlify.app/), with a mirror on **GitHub Pages** at [vigneshyadala.github.io/portfolio](https://vigneshyadala.github.io/portfolio/). Both deploy automatically from the `main` branch — any push updates the live sites.
